# Braavo Design System

## Brand Identity
- **Product**: Braavo — AI-powered SDLC platform
- **Tone**: Professional, clean, focused on productivity
- **Audience**: Product Managers, Designers, Developers

## Colors
| Token | Value | Usage |
|-------|-------|-------|
| primary-500 | #3b82f6 | Primary actions, links |
| primary-900 | #1e3a8a | Headings, emphasis |
| secondary-500 | #64748b | Secondary text |
| success | #10b981 | Success states |
| warning | #f59e0b | Warnings |
| error | #ef4444 | Errors |
| surface | #f8fafc | Card backgrounds |
| background | #ffffff | Page background (light) |
| background-dark | #0f172a | Page background (dark) |

## Typography
- **Font**: Inter (sans-serif), Fira Code (monospace for code)
- **Scale**: xs:12px, sm:14px, base:16px, lg:18px, xl:20px, 2xl:24px

## Spacing
- Base unit: 4px
- Common: 8px, 16px, 24px, 32px, 48px

## Components
- Rounded corners: 8px (cards), 6px (buttons), 4px (inputs)
- Shadows: subtle, layered (sm, md, lg)
- Buttons: solid primary, outline secondary, ghost tertiary

## Layout
- Max content width: 1280px
- Sidebar: 280px fixed
- Mobile breakpoint: 768px

## Accessibility
- WCAG 2.1 AA compliant
- Minimum contrast 4.5:1
- Focus rings on interactive elements
